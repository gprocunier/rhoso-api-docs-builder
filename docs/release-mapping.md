# Release Mapping

| RHOSO | OpenStack | Name | Status |
| --- | --- | --- | --- |
| 18.0 | 2023.1 | Antelope | supported |
| 19.0 | 2026.1 | Gazpacho | beta |

## RHOSO 18.0

RHOSO 18.0 generation is enabled. The supported-service catalog is derived from
the RHOSO 18.0 planning documentation's service operator table and known
limitations section.

## RHOSO 19.0

RHOSO 19.0 appears in the site release picker as beta. Supported API generation
is blocked until public Red Hat documentation provides a supportable service list
for the release.
